Definitions

General

  1. IDE - Integrated Development Environment
  2. Server - A software program that responds to requests made by browsers.
  3. Browser - A software program that sends requests to a server and receives responses in return.
  4. HyperText Transfer Protocol (HTTP) - the rules for communicating between server and browser. 
    GET /index.html HTTP/1.1
Host: c9.io
User-Agent: Firefox/5.0 
Accept: text/html
Accept-Language: en-us
Accept-Charset: ISO-8859-1,utf-8;
Cookie: PHPSESSID=r2t5uvjq435r4q7ib3vtdjq120
Keyboard shortcuts

Files

  1. CTRL+o - Open a file.
  2. CTRL+s - Save all files.

Text

  1. CTRL+a - Select ALL text in the open file.
  2. CTRL+c - Copy the text that's been highlighted.
  3. CTRL+x - Cut the text that's been highlighted.
  4. CTRL+v - Paste the text that's in the clipboard.
  5. CTRL+z - Undo.
  6. TAB - Indent highlighted text.
  7. SHIFT+TAB - Un-indent highlighted text.

Web Browser

  1. CTRL+TAB - switch between open tabs in IDE.
  2. CTRL+r - Reload web page in browser.
